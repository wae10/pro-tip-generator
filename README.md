# Pro Tip Generator
## Description
Oddly specific, yet personally useful organization tool that creates a separate Word Document containing only the most important information (aka 'pro tips') extracted from an existing file of notes.

## Why
When taking notes for class, I sometimes catch a rare but important announcement or piece of information from the professor that gets lost in the rest of my daily notes. I only review my notes before exams, whereas these particular announcements require more urgent attention. Therefore, my two problems are 1) how should I divide my notes? And 2) how can I remember to do it?

The Pro Tip Generator is my solution. It automates an otherwise tedious process of manually searching through each class notes document for the necessary content and creating new files containing that content. All I need to do is type the phrase 'pro tip' before any random, important note, and the Python script will do the rest.
